# mental-health-portal
A full-stack web application designed to provide accessible mental health support through therapy sessions, self-help tools, and AI-powered assistance. This platform helps users manage stress, anxiety, and emotional well-being in a safe and user-friendly environment.

🚀 Features
🔐 User Authentication
Secure signup and login using JWT
Protected user sessions
🧑‍⚕️ Therapist Booking
Browse available therapists
Book and manage therapy sessions
💬 AI Chat Support
Get instant mental health guidance using AI chatbot
24/7 emotional assistance
📊 Mood Tracking
Track daily mood and emotional patterns
Visual insights for self-awareness
📁 Session History
View past sessions and notes
Track personal progress over time
📱 Responsive UI
Clean and accessible design across devices
